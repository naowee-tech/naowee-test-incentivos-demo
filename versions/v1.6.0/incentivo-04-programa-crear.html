Datos del programa

Define los datos básicos del programa. Esto lo lanza el ministerio y sirve como contenedor de todo: rubro, tipos, condiciones y códigos.

1 Información general

Aparecerá en listados, reportes y notificaciones.
Útil para reportería y trazabilidad.

2 Vigencia y cobertura territorial

Nacional Atlántico Cundinamarca Antioquia Valle Santander + Agregar región
Selecciona dónde aplica el programa. "Nacional" incluye todas las regiones.

Rubro presupuestal

Define el monto de dinero asignado por el ministerio a este programa. El sistema descontará automáticamente de este rubro por cada asignación realizada.

1 Monto asignado por el ministerio

En pesos colombianos (COP). El sistema descuenta de este monto por asignación.

2 Recordatorio

Según las reglas de negocio acordadas, un programa sólo puede tener un rubro. Si posteriormente el ministerio amplía el presupuesto, se aumenta este rubro y se cargan códigos adicionales consistentes con el valor unitario.

Tipos de incentivo

Define qué incentivos entregarás en este programa. Puede ser un único tipo (ej. beca) o varios (ej. beca + kit). Cada tipo puede ser cualitativo, monetario o ambos.

1 Incentivo #1

Cualitativo con valor monetario
Tiene nombre descriptivo (ej. "beca") y un valor asociado (ej. $1.000.000). Ideal cuando quieres trazabilidad del monto entregado.
Sólo monetario
Valor puro sin categoría descriptiva. Ej. "dinero · $500.000".
Sólo cualitativo
Sin valor monetario explícito. Ej. "pase al parque", "ingreso al evento". No se descuenta del rubro.
Pesos por incentivo entregado.
Cantidad de beneficiarios planeados (informativo).

Condiciones de elegibilidad

Parametriza las reglas que debe cumplir un deportista para aparecer en los resultados de búsqueda del operador. Puedes combinar varias condiciones con AND / OR.

1 Constructor de reglas

Construye el árbol lógico de condiciones. Ejemplo: (edad ≥ 18 AND categoría = prejuvenil) OR (medalla = oro).

GRUPO 1
Todas las condiciones de este grupo deben cumplirse
GRUPO 2
Grupo alternativo (OR con grupo 1)
Vista previa de la regla:
(edad ≥ 18 AND categoria = "Prejuvenil o superior") OR (logros ∈ {"oro","plata"})
